# digitaljesse.com

Personal portfolio website for Jesse Johnson.

## Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Hosting:** Vercel
- **Forms:** Formspree

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view.

## Deploy

Push to main branch. Vercel auto-deploys.
